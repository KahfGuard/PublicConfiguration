# KahfGuard Public Configuration

Public configuration files and assets for the KahfGuard DNS filtering system.

## Contents

### IP Lists
**`lists/ip/`** - IP address allow/block lists for rate limiting and access control
- `allow.txt` - Whitelisted IP addresses
- `block.txt` - Blocked IP addresses

### Router Configuration
**`router-config/`** - Network router configurations

### Assets
**`banner/`** - Visual assets and banners
**`voices/`** - Audio files (multi-language)

### Application Configurations
**`configs/`** - Configuration files for KahfGuard services

### Releases
**`releases/`** - Release artifacts and versioned builds

### Web Resources
- `index.html` - Public landing page
- `windows.json` - Windows-specific configuration

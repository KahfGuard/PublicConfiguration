# SafeSearch Enforcement Configuration
# Format: DOMAIN SAFESEARCH_ENDPOINT_HOSTNAME
#
# When a client queries for any domain in this list, DNSDist responds with
# the IP address of the SafeSearch enforcement endpoint (spoofed DNS response)
#
# The system resolves each endpoint hostname to IPv4/IPv6 addresses at startup
# and creates DNS spoof rules for all domains mapping to that endpoint

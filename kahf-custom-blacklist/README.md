# Kahf Custom Blacklist

Domain blacklists for Islamic values-based content filtering.

## Categorization Guide

### adult.txt
Pornography, nudity, explicit content, sex stories, adult blogs, erotic material, NSFW sites

### gambling.txt
Casino, betting, lottery, poker, slots, sports betting, dice games, raffles, wagering, jackpot

### dating.txt
Dating apps, matchmaking, hookup sites, chat roulette, random video chat, relationship platforms

### violence.txt
Violence, gore, weapons, extremism, terrorism, hate speech, self-harm, suicide, blasphemy, anti-religious content

### malware.txt
Viruses, phishing, scams, fraud, spam, malicious downloads, suspicious sites, fake alerts

### piracy.txt
Torrents, illegal downloads, cracked software, pirated movies/TV, free streaming, copyright violation

### social.txt
User-generated content, forums, social platforms, gaming sites, media hosting (mixed content requiring review)

## File Format

- One domain per line
- Alphabetically sorted
- No duplicates
- No empty lines

Example:
```
badsite1.com
badsite2.org
subdomain.badsite3.net
```

## Adding New Domains

1. Identify the primary content type using keywords above
2. Add to appropriate file
3. Keep alphabetically sorted
4. Remove duplicates
